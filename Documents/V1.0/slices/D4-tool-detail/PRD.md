## 1. 这块做什么

在资产 Tab 中点击一个工具后，右侧主区变为工具的代码视图。这是用户查看、编辑、测试工具的主要界面。

---

## 2. 布局

在资产 Tab 中，选中工具时：

```
[资产列表（左面板）] | [对话列表或对话内容] | [工具代码视图（右半区）]
```

工具代码视图占据右侧主区的右半部分（如果没有打开任何对话，则占满整个主区）。

---

## 3. 工具代码视图

```
┌──────────────────────────────────────────────────────┐
│ 📦 发送邮件          ✅已测试   [导出]  [删除]        │
│ 通过 SMTP 发送邮件   分类：邮件                        │
│ ──────────────────────────────────────────────────── │
│ [代码]  [参数]  [测试]                                 │
│ ──────────────────────────────────────────────────── │
│                                                      │
│  （Tab 内容区，占满剩余高度）                          │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 3.1 代码 Tab

- Monaco Editor，Python 语法高亮
- 默认只读；点击"编辑"按钮切换为可编辑
- 编辑后有"保存"和"取消"，保存时做语法检查
- 语法错误不允许保存，显示具体错误位置

### 3.2 参数 Tab

从代码函数签名解析出的参数表格：

| 参数名 | 类型 | 必填 | 说明 |
|---|---|---|---|
| to | string | 是 | 收件人邮箱 |
| subject | string | 是 | 邮件主题 |

### 3.3 测试 Tab

- 动态表单（从参数 schema 生成输入框）
- "▶ 运行测试"按钮
- 测试结果实时显示（success/error + 输出内容）
- 测试历史列表（最近 20 条，按时间倒序）

---

## 4. 工具被工作流引用时的提示

若该工具已被工作流使用：保存代码时在顶部显示一条警告横幅：  
`"此工具已被 N 个工作流使用，修改后建议重新测试。"` （不阻止保存）

---

## 5. 验收标准

- [ ] 在资产 Tab 点击工具，右半区出现代码视图
- [ ] 代码 Tab：语法高亮，点"编辑"可修改，保存有语法检查
- [ ] 参数 Tab：从代码签名正确解析参数列表
- [ ] 测试 Tab：填参数 → 运行 → 结果实时显示 → 记录到历史
- [ ] 测试历史最多显示 20 条
- [ ] 工具被引用时保存显示警告
