## 1. 这块做什么

工作流画布是资产 Tab 右侧主区的右半部分（当选中工作流时）。它以可视化方式呈现工作流的节点和连线，是用户理解和操作工作流结构的主界面。

---

## 2. 画布基础功能

### 2.1 视图操作
- **缩放**：鼠标滚轮缩放（10% - 200%），Ctrl+0 复位
- **平移**：按住空格拖拽，或触控板双指滑动
- **全屏适配**：点击右下角"适应画布"按钮，自动缩放到全部节点可见

### 2.2 节点操作
- 节点可拖拽移动（自动吸附到网格）
- 节点单击：选中（高亮边框），再次单击：展开节点配置面板
- 节点右键菜单：删除、复制、禁用/启用
- 节点配置面板：从画布底部滑出（不遮挡画布），显示该节点的输入输出配置

### 2.3 连线操作
- 从节点的输出端口拖拽连到另一节点的输入端口
- 点击连线：选中（变蓝）
- 选中连线按 Delete：删除连线
- 不合法的连接（如循环依赖）自动拒绝

### 2.4 节点状态颜色

| 状态 | 颜色 | 含义 |
|---|---|---|
| 正常（已配置）| 蓝色边框 | 配置完整，可运行 |
| 缺失工具 | 黄色边框 | 引用的工具尚未创建 |
| 运行中 | 蓝色闪烁 | 当前正在执行 |
| 成功 | 绿色边框 | 本次运行成功 |
| 失败 | 红色边框 | 本次运行失败 |
| 禁用 | 灰色 | 被用户禁用，跳过执行 |

### 2.5 工具栏（顶部）

```
[工作流名称]  [草稿]   ▶ 运行一次  [部署]  [运行历史]
```

---

## 3. 画布与对话的关系

- 用户在对话中描述需求 → AI 生成/更新节点 → 画布实时刷新
- 用户在画布上的操作（移动节点、改连线）→ 静默保存到 DB
- 下一次用户发消息时，当前工作流状态作为隐藏上下文注入 AI

---

## 4. 小地图

右下角有小地图（minimap），显示整个工作流的全局视图，点击可快速跳转到指定位置。
